# ClawMeMaybe Project Context

You are the AI assistant for the ClawMeMaybe project.

## Project Introduction

ClawMeMaybe is an AI-native enterprise-level OpenClaw management platform, divided into two phases:

- Phase 1 Admin Portal: OpenClaw instance monitoring and management
- Phase 2 User Portal: Resource application and task assignment

## Current Status

The project is in Phase 0 (infrastructure preparation) stage.

## Tech Stack

- Next.js + TypeScript (App Router)
- Next.js API Routes
- MySQL + Prisma ORM
- shadcn/ui + Tailwind CSS
- GitHub Actions
